遇到问题

执行，hexo d报错：

ERROR Deployer not found: git

解决方案

这是因为没安装hexo-deployer-git插件，在站点目录下输入下面的插件安装就好了：

npm install hexo-deployer-git –save

如果还没解决：设置线sshkey
